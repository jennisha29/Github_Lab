Airflow Lab 01 – Dashboard

Overview

In this lab, Apache Airflow is used to run a complete machine learning workflow end to end. The pipeline takes the e-commerce customer data and groups similar customers together using KMeans clustering. And, it automatically finds the best number of clusters using the elbow method.

Load Data → Preprocess → Train Model → Predict → Visualize

Dataset Used:

  • Training dataset: ecommerce_customers.csv
  • Test dataset: test_ecommerce.csv
  • Features used for clustering: TOTAL_SPEND, BROWSE_SESSIONS, LOYALTY_POINTS

What was done in this lab:

  • Built an Airflow DAG to run the entire workflow step by step
  • Loaded the e-commerce customer dataset
  • Cleaned and scaled the data for clustering
  • Trained a KMeans model and used the elbow method to find the best number of clusters
  • Saved the trained model and clustering results
  • Generated visual outputs like the elbow curve and cluster scatter plot
